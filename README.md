## Portfolio — React + Vite

Одностраничное портфолио‑сайт фронтенд разработчика на **React + Vite** с акцентом на аккуратный UI, модалки и адаптивную верстку.

---

## Стек

- **React 19** + **Vite**
- **JavaScript (ESNext)**
- **CSS** (кастомная верстка, без UI‑библиотек)

---

## Структура проекта

- `src/main.jsx` — вход в приложение, монтирование `App`
- `src/App.jsx` — композиция основных секций страницы
- `src/App.css` — стили компонентов страницы (шапка, блок “обо мне”, портфолио, модалки и т.д.)
- `src/index.css` — глобальные стили, фон и свечение
- `public/` — картинки для экрана портфолио, галереи и иконки соцсетей

---

## Основные блоки

- **Header** — логотип и навигация по странице
- **Hero / обо мне** — заголовок, краткий текст, фото и кнопка “Заказать”
- **Portfolio** — пример экрана с аккордеоном цен
- **Gallery** — превью работ
- **Footer** — юридический текст и копирайт
- **Модальные окна**:
  - форма “Отправить заявку”
  - модалка с контактами (VK / Telegram)

Оба модальных окна открываются по клику и скрываются через условный рендеринг.

---

## Запуск проекта

```bash
npm install
npm run dev
```

После запуска Vite выведет ссылку, обычно `http://localhost:5173/`.

---

## Билд

```bash
npm run build
npm run preview
```

`build` собирает оптимизированную версию в папку `dist`, `preview` поднимает локальный сервер для проверки.
